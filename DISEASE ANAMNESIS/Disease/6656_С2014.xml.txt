  Артериальная гипертензия более 5 лет. Максимальные цифры АД 150/100мм.рт.ст. Привычные 140/90мм.рт.ст. Без гипотензивной терапии. С 2014г беспокоят нетипичные кратковременные боли в локтевом сгибе левой руки в утренние часы без связи с ФН 1-2 раза в месяц.. К врачам не обращался, не обследовался. ФН переносил удовлетворительно.  15.05.2014 в 7.00 имел место эпизод интенсивных болей в локтевом сгибе левой руки, возникший внезапно, без связи с ФН. Вызвал бригаду СМП. на ЭКГ: Элевация ST c V1 по V5 с формированием зубцов Q V1 по V6, отрицательными зубцами Т c V2 по V6. Даны нагрузочные дозы антиагрегантов (аспирин 250мг, плавикс 300мг), гепарин 5000ЕД, анаприлин 40мг, боли купированы введением нитратов (около 1,5мг). В экстренном порядке доставлен бригадой СМП в ОАиР№2 с представлением об остром трансмуральном ИМ передне-боковой локализации от 15.05.2014. На момент поступления ангинозные боли, боли в левой руке отрицает. На ЭКГ: картина переднего распространенного ИМ: Элевация ST c V1 по V5 с формированием зубцов Q V1 по V6, отрицательными зубцами Т c V2 по V6. Определены показания к экстренной КАГ, с решением вопроса о необходимости реваскуляризации. Абсолютных противопоказаний нет. Информированное согласие получено. Группа крови А (II), RH (+). Дано: плавикс 300мг, аспирин 100мг, лосек 40мг, гепарин 5000ЕД.
