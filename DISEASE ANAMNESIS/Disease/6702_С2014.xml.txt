  АД ранее не измерял, не контролировал. Артериальную гипертензию отрицает. В течение последних двух недель впервые в жизни начали беспокоить боли давящего характера при ФН. К врачам не обращался, не обследовался.  16.05.2014 в 00.00 внезапно возникли интенсивные давящие боли за грудиной. Вызвал бригаду СМП. На ЭКГ зарегистрировано: Элевация ST c V1 по V6 с формированием зубцов Q V3 по V6, начинающими формироваться отрицательными зубцами Т c V1 по V6. Даны нагрузочные дозы антиагрегантов (аспирин 250мг, зилт 300мг), гепарин 5000ЕД, анаприлин 40мг, боли купированы введением нитратов. 16.05.2014 доставлен экстренно бригадой СМП в ОАиР№2 с представлением об остром передне-боковым распространенным трансмуральным ИМ от 15.05.2014. На момент поступления ангинозные боли отрицает. На ЭКГ: картина переднего распространенного ИМ: Элевация ST c V1 по V6 с формированием зубцов Q V4 по V6, начинающими формироваться отрицательными зубцами Т c V1 по V6, реципрокными изменениями по нижней стенке (депрессия ST в II, III, AVF). Определены показания к экстренной КАГ, с решением вопроса о необходимости реваскуляризации.
