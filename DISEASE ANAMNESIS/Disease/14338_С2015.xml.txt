 Длительный анамнез ГБ, адаптирован к АД 120/80 ммртст, максимальные цифры АД 180/100 мм.рт.ст., постоянно принимает эпросартан, бисопролол. Дебют ИБС 06-07 июня 2015г. - появления ангинозных приступов. 09.06.15г. - переносит непроникающий ИМ нижне - боковой локализации, в условиях ФМИЦ выполнена КАГ с одномоментной тромбэкстракцией из МА, коронаропластикой со стентированием МА, имплантирован 1стент без л/п. Реваскуляризация неполная, обсужден с рентгенхирургом Козулиным В.Ю., показано выполнение стресс ЭХоКГ в плановом порядке (через 4-6 мес.) для решения вопроса о дальнейших этапах реваскуляризации. Выписан в удовлетворительном состоянии, со слов рекомендаций придерживался. 25.06.15г- повторная госпитализация в ФМИЦ с представлением о нестабильной стенокардии, данных за ОКС не получены, выписан под наблюдение кардиолога по м/ж. После отмечал "дискомфорт" в грудной клетке на фоне бытовых ФН. 18.07.15г- интенсивный ангинозный приступ, нарядом СМП госпитализирован в ОАР ФМИЦ с представлением об ОКС. Уровень тропонина в пределах нормы. По ЭКГ- без острой очаговой патологии. 20.07.15г- выполнена КАГ - ПМЖА- без г/з стенозов, ДА- устьевой стеноз до 70-75%, ОА- стент без признаков рестеноза/тромбоза, ПКА- стеноз до 90%, нельзя исключить тромбоз. Одномоментно выполнена коронаропластика со стентированием ПКА (1 стент без ЛП). Интраоперационно без осложнений. Реваскуляризация неполная, рекомендовано стресс-ЭхоКГ через 4-6 нед для решения вопрос о функциональной значимости устьевого стеноза ДА. С учетом клиники, лабораторно-инструментальных методов обследования, данных за ОИМ не получено. Ангинозные боли не рецидивировали, гемодинамика стабильна, не лихорадит. Периодически отмечает дискомфорт в эпигастральной обл. Переведен в удовлетворительном состоянии в ЛРК для дальнейшего лечения и реабилитации.
